# Step 4: Polish

Rewrite the article to remove AI tone and maximize impact.

## Improve

### Rhythm
- Vary sentence length
- Short punchy sentences for emphasis
- Longer sentences for explanation

### Punchlines
- Make key sentences quotable
- Suitable for screenshots
- Memorable enough to repeat

### Readability
- Break long paragraphs
- Add visual breaks where needed
- Ensure flow between sections

## Add

- **2 memorable sentences** suitable for screenshots/sharing
- **1 strong concluding line** that sticks

## Remove

- Filler phrases
- Redundant transitions
- Weak qualifiers (过多的"可能"、"或许")
- Any remaining AI-speak

## Check

Before output, verify:
- [ ] No "随着/近年来" type openings
- [ ] No "首先其次最后" structure
- [ ] At least one specific number or example
- [ ] At least one clear judgment (not just "可能会")
- [ ] Opening hooks in 2 sentences
- [ ] Ending is strong, not trailing off

## Do NOT

- Change the core argument
- Add new information
- Make the tone more "professional" (keep it human)
- Expand length unnecessarily

## Output

Return the polished article only, ready to publish.
