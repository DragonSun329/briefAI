# Step 3: Write

Write a publishable article using the thesis.

## Rules

### Opening
- Start with a personal or experiential statement
- **Never** start with "Recently" / "最近" / "随着" / "据报道"
- Hook the reader in the first two sentences
- Make them curious, not informed

### Body
- Explain the event simply, then interpret it
- Use concrete examples (numbers, names, specifics)
- Every section must add a new insight
- Build your argument progressively
- Anticipate objections and address them

### Ending
- End with a strong statement or implication
- What should the reader think or do differently?
- Leave them with something to remember

### Structure Guide
1. Hook (1-2 sentences)
2. Setup context (brief)
3. Core argument (main body)
4. Supporting evidence / examples
5. "So what" implications
6. Strong closer

## Constraints

- **Length:** 1500–2200 Chinese characters
- **Tone:** Clear, confident, human. Not academic.
- **Voice:** First person when appropriate ("我" not "笔者")
- **Format:** Short paragraphs (3-4 lines max for WeChat readability)

## Anti-AI Smell

Avoid:
- "首先、其次、最后" structure
- "值得注意的是"
- "不难看出"
- "综上所述"
- "让我们拭目以待"
- Excessive hedging ("可能"、"或许" everywhere)
