# Style Guide

## Opening Rules

**Never start with:**
- "最近..."
- "据报道..."
- "随着AI的发展..."
- "在当今时代..."
- Any generic setup

**Always start with:**
- A personal observation
- A surprising fact
- A provocative statement
- A concrete moment/scene

## Structure

1. **Hook** (1-2 sentences) — 抓住注意力
2. **Setup** (2-3 sentences) — 交代背景，但要有观点
3. **Core Argument** (主体) — 你真正想说的事
4. **Evidence/Examples** — 支撑论点的具体案例
5. **Implication** — 这对读者意味着什么
6. **Closer** — 一句话收尾，要有力

## Tone

- Confident, not arrogant
- Curious, not know-it-all
- Direct, not aggressive
- Technical depth when needed, accessible by default

## Formatting (WeChat)

- 段落短，3-4行换一次
- 重要观点**加粗**
- 适当用 emoji 做视觉断点（别滥用）
- 可以用引用块做金句
- 不用 markdown 标题符号（# ## ###）

## Length

- 目标：1500-2200 字
- 太短没深度，太长没人看完

## Anti-AI-Smell Checklist

- [ ] 没有"首先、其次、最后"结构
- [ ] 没有"值得注意的是"
- [ ] 没有"不难看出"
- [ ] 没有"综上所述"
- [ ] 开头不是"随着/近年来/在...背景下"
- [ ] 有至少一个具体数字或案例
- [ ] 有至少一个明确判断（不是"可能会"）
